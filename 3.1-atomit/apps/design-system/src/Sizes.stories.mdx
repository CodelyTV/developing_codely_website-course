# Sizes

```scss
@use "@codely/design-system/src/settings/sizes";
```

We use sizes for media queries. We currently target the following sizes:

```
$desktop: 1440px;
$tablet: 1024px;
$mobile: 480px;
$mobile-xs: 375px;
```
