## Heading sizes

For heading sizes we use mixins to be able to change the size with media queries easily.

```scss
@use "@codely/design-system/src/tools/typography";
```

Note: if you use body sizes and heading sizes in the same component the namespaces clash, so you can use an alias, for example:

```scss
@use "@codely/design-system/src/tools/typography" as mixins;
```

`@include mixins.h500-regular;`

<div className="h500-regular">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</div>
